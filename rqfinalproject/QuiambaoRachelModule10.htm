Recommendations

1.       HTML Recommendation for “A charset attribute on a meta element found after the first 1024 bytes.”

a.       When looking at the site’s HTML, the meta charset attribute was added after a long string of text. I would move this to the beginning of the <head> section.

2.       HTML Recommendation for “Trailing slash on void elements has no effect and interacts badly with unquoted attribute values.”

a.       I would remove the trailing slash on the link and meta elements.

3.       CSS Recommendation for “Value Error : max-width auto is not a max-width value : auto”

a.       ‘Auto’ is not an allowed value for max-width. I would change it to a percentage value.

4.       CSS Recommendation for “Value Error : font-size normal is not a font-size value : normal”.

a.       I would suggest using pixel sizes for this, or if they want to use keywords, they can use “small” or “medium”.

5.       Siteimprove Recommendation for “Image link is missing alternative text”

a.       I would add an alt tag to the image with appropriate text describing the image in the page

6.       SiteImprove Recommendation for “Link identified only by color”

a.       The link only shows when you hover over it, and it changes color. To make it more accessible, I would underline the link or add a button.
